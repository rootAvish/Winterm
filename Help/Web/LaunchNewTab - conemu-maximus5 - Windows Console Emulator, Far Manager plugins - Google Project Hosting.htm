Create new tab from existing one

Use -new_console switch while executing command. For example, if you want to start PuTTY in new ConEmu tab, type in command prompt (cmd.exe started in ConEmu tab)

and it will be launched in the new tab of ConEmu.

-new_console works even in batch files.
